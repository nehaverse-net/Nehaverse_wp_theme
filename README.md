# Nehaverse WordPress Theme

既存の Nehaverse サイトデザインを WordPress テーマ化したものです。

## インストール

1. `wordpress-theme/nehaverse` フォルダを WordPress の `wp-content/themes/` に配置します。
2. 管理画面の「外観 > テーマ」から `Nehaverse` を有効化します。
3. 「設定 > パーマリンク」で投稿名形式などを保存します。

## ヘッダーナビ

ヘッダーのナビは「外観 > メニュー」で変更できます。

1. 新しいメニューを作成します。
2. 固定ページ、カテゴリー、カスタムリンクなど、表示したい項目を追加します。
3. メニューの位置で `ヘッダー項目 1` から `ヘッダー項目 6` のどれかを選択して保存します。

メニュー未設定時は、テーマ標準の「ホーム」だけが表示されます。

`ヘッダー項目 1` から `ヘッダー項目 6` に割り当てたメニューは、それぞれのメニュー名がヘッダー上の親項目として表示されます。メニュー内に追加した固定ページ、カテゴリー、カスタムリンクは、その親項目のドロップダウン内に表示されます。

例:

- `Minecraft` というメニューを作る → `ヘッダー項目 1` に割り当てる
- `参加方法` というメニューを作る → `ヘッダー項目 2` に割り当てる
- `役立ち情報` というメニューを作る → `ヘッダー項目 3` に割り当てる

右端の「コミュニティ参加」ボタンは「外観 > カスタマイズ > Nehaverse Header」から文言、URL、新しいタブで開くかどうかを変更できます。

## 推奨ページ

次の固定ページを作成し、スラッグを合わせてください。テーマ内で専用テンプレートが自動適用されます。

- `minecraft`
- `rules`
- `products`
- `contact`

トップページは `front-page.php` で表示されます。「設定 > 表示設定」で固定ページをホームページに指定してください。

## お知らせ

テーマ有効化後、管理画面に「お知らせ」投稿タイプが追加されます。トップページのNews欄には最新5件が表示されます。

カテゴリ風の表示を変えたい場合は、お知らせ投稿のカスタムフィールド `news_type` に次のいずれかを設定してください。

- `maintenance`
- `event`
- `update`
- `other`

## お問い合わせフォーム

Discord Webhook 連携を使う場合は `wp-config.php` に次を追加してください。

```php
define('NEHAVERSE_DISCORD_WEBHOOK_URL', 'https://discord.com/api/webhooks/...');
```

未設定の場合、フォーム送信はエラー表示になります。

## サービス稼働状況

トップページ最下部の参加CTAとサイトフッターの間に、WebサイトとMinecraftサーバーの稼働状況が表示されます。

「外観 > カスタマイズ > Service Status」の `Status API URL` に、Cloudflare Workerの `/api/status` エンドポイントを設定してください。

例:

```text
https://nehaverse-status.example.workers.dev/api/status
```

表示は60秒ごとに自動更新されます。API URLが未設定の場合は、設定が必要であることを示すメッセージが表示されます。
